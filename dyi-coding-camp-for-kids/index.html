I had already put together a Raspberry Pi(rpi) for my home network (which is a blog post on its own). So I thought that this would be an excellent solution for my boys linux rig as well.

I found a BOGO offer on Pis from Arrow Electronics. While you can just buy the entire kit from Amazon, these did not include the fans, case, or power adapter. So if your just looking for a case kit, Amazon has those too.

So at the end of the day, you will need:

  • Raspberry Pi
  • Case
  • Fan and Heatsinks(optional)
  • Power Adapter
  • SD Memory Card (8GB Minimum)
  • Monitor
  • Keyboard and mouse

Software

In order to get an operating system on the Raspberry Pi, you will need an SD Memory card.

The os is over 4GB so they recommend a minimum of 8GB. However, Amazon has 32GB memory card for under $10.
