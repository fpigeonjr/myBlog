Some of the things I use

December 11, 2018

Editor + Terminal

I have been using Visual Studio Code as my code editor for over a two years now and have been very happy with it. Probably my favorite electron app by far. My favorite color themes are Cobalt2, Dracula, Material Facebook. My favorite font is [Dank Mono][dank mono].

For my terminal, I have stuck with iTerm2. It is reliable, customizable, and just works. I go fullscreen with a slight transparency. I also use oh-my-zshell, pure-prompt, with the facebook theme. I played around with Hyper but had some issues so I switched back.
